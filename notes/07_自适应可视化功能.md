# 自适应过程可视化功能

## 📋 功能概述

新增了**学习分析**页面，实时展示自适应算法的决策过程和学习效果，让学生和面试官都能直观地看到算法的工作原理。

## 🎯 核心功能

### 1. 自适应状态卡片
展示当前的自适应决策状态：
- **当前难度**：最近一题的难度等级
- **最近3题均分**：滑动窗口的平均分数
- **稳定性**：是否连续3题在同一难度（避免频繁跳级）
- **下一题难度**：算法建议的下一题难度
- **决策依据**：详细的决策理由（包含分数、稳定性、知识点掌握度）

### 2. 答题趋势图表
使用 Chart.js 绘制双轴折线图：
- **左轴（蓝色）**：得分趋势（0-100分）
- **右轴（橙色）**：难度变化（简单/中等/困难）
- **交互提示**：鼠标悬停显示具体知识点

### 3. 知识点掌握度
展示每个知识点的学习情况：
- **掌握等级**：熟练掌握(≥85) / 基本掌握(≥70) / 需要练习(≥60) / 未掌握(<60)
- **进度条**：可视化显示平均分
- **练习次数**：该知识点的答题数量

## 🔧 技术实现

### 后端接口

**新增接口**：`GET /student/adaptive_stats`

**参数**：
- `student_id` (必填)：学生ID
- `keyword` (可选)：指定知识点

**返回数据结构**：
```json
{
  "status": "success",
  "data": {
    "recent_trend": [
      {
        "time": "01-15 10:30",
        "score": 85,
        "difficulty": "中等",
        "keyword": "递归"
      }
    ],
    "keyword_stats": {
      "递归": {
        "count": 5,
        "total_score": 425,
        "avg_score": 85.0,
        "level": "熟练掌握"
      }
    },
    "adaptive_state": {
      "current_difficulty": "中等",
      "avg_score_last_3": 85.0,
      "is_stable": true,
      "next_difficulty": "困难",
      "reason": "最近3题平均分: 85.0 | 难度稳定性: 稳定 | 建议调整: 中等 → 困难"
    }
  }
}
```

## 📊 算法可视化逻辑

### 滑动窗口
```
最近10题 → 取最近3题 → 计算平均分 → 判断是否升级/降级
```

### 稳定性检查
```
最近3题难度 → 检查是否一致 → 稳定才允许升级
```

### 知识点维度
```
同一知识点的历史记录 → 计算平均分 → 影响难度判断
```
