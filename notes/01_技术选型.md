# 技术选型

## 大语言模型
系统核心能力基于大语言模型的自然语言理解与生成能力，用于题目生成与开放性问题的智能评分。可选模型包括 OpenAI GPT 系列或国产大模型接口。

## 开发工具：vscode，miniconda，git


## 后端技术
后端采用 Python 语言开发
选用FastAPI框架，实现业务逻辑处理、接口服务及模型调用。(并发能力强：async)

## 前端技术
前端采用Vue框架，构建教师端和学生端界面，实现测评配置、在线作答和结果反馈。

## LLM 应用框架
使用 LangChain 框架构建检索增强生成（RAG）流程，组织文档加载、向量检索与 Prompt 调用。

## 数据存储
- 关系型数据库：用于存储用户信息、测评记录等结构化数据  
- 向量数据库：采用 ChromaDB 存储文本向量，用于语义检索
